# DartsTool

ダーツの得点計算・練習ツール

## 🎯 アレンジクイズ

ハードダーツ（スティールティップ）の01ゲーム用アレンジ計算クイズWebアプリです。

### 機能

- **ダブルアウト専用**: ハードダーツのルールに準拠し、ダブルアウトのみ
- **アレンジ表**: 2〜170点のアレンジデータを収録
- **優先フィニッシュダブル**: D16, D12, D14, D18, D20を優先
- **キーボードUI**: 左側に倍率キー(×1/×2/×3/BULL)、右側に1〜20の数字キー(5列)
- **複数ルート対応**: 正解ルートが複数ある場合はすべて表示
- **Tips表示**: アレンジの理由を解説
- **モバイル対応**: スマートフォンで快適に利用可能

### 使い方

1. `index.html` をブラウザで開くか、Webサーバーでホストしてください
2. 表示された残りスコアに対するアレンジを入力します
3. 左側の倍率キー（×2でダブル、×3でトリプル）を選択してから数字キーを押します
4. BULLキーでアウターブル(25)、×2+BULLでダブルブル(50)を入力できます
5. 「回答」ボタンで正誤判定。正解ルートとTipsが表示されます

### デプロイ

静的HTMLファイルのみで構成されているため、GitHub Pages等で簡単にホスティングできます。
